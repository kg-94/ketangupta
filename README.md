## 🚀 Getting Started

### Local Development

1. Clone the repository
2. Open `index.html` in your browser to view the home page
3. Navigate to `cv.html` for the full CV

### Deployment

This site is hosted on GitHub Pages. To deploy:

```bash
git add .
git commit -m "Update CV portfolio"
git push origin master
```

The site will be live at `https://ketangupta.github.io` (or your custom domain if configured).

## ✏️ Customization

### Update Content

- **Home Page**: Edit text in `index.html` within the hero, about, and contact sections
- **CV Page**: Update experience, skills, projects, and education in `cv.html`

### Change Colors

Edit CSS variables in `styles.css`:
```css
:root {
    --primary: #6366f1;      /* Indigo */
    --secondary: #ec4899;    /* Pink */
    --accent: #f59e0b;       /* Amber */
}
```
